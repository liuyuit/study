# 加权无向图的数据类型

加权无向图应该如何表示？对于邻接表来说，可以在链表中添加一个权重域。但我们可以用另一种更加通用的API来表示Edge对象。

加权边的API

| Edge   |                |
| ------ | -------------- |
| Edge   | 初始化         |
| weight | 边的权重       |
| either | 边的顶点之一   |
| other  | 边的另一个顶点 |
