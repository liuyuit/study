## 十二进制的补数

我们想要从6点调到3点，我可以直接往回调3个刻度，也可以往前调9个刻度。

这就反映了如果忽略进位，那么6 - 3 = 6 +9。

```
6 - 3 = 6 +9
6 - 3 = 6 + 9 - 12
```



## 十进制中补数

### 如何求一个数的补数

如果两个数加起来刚好进一位，那这两个数互为补数

如果是一个三位的数直接用1000减去这个数就可以,但是为了避免进位。求319的补数可以这样

```
1000-319
=999-319+1
=681
```



### 利用补数将减法转换为加法

减去一个数可以等于加上这个数的补数然后忽略进位，例如计算6-3

```
6-3
=6-(10-7)
=6+7-10
```
