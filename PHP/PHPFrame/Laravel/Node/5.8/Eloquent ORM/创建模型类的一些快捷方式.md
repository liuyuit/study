## 生成模型类

```
php artisan make:model Models/Main/Card
```

此命令会生成文件 `app/Models/Main/Card.php`。

```
<?php

namespace App\Models\Main;

use Illuminate\Database\Eloquent\Model;

class Card extends Model
{
    //
}
```
